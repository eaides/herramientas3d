# Summary

I recently needed some PCB standoff spacers for an Arduino project so I created these simple customizable ones in **OpenSCAD** as a learning exercise.  I print a few now when I need them.  

The source for this project is maintained on GitHub: https://github.com/mseminatore/mdsSpacer

Tips appreciated!



# Print Settings

Printer Brand: 3D Systems
Printer: Cube 3
Rafts: No
Supports: No
Resolution: 200
Infill: 100%

Notes: 
I use low resolution but 100% since these are small.  You can probably use a lower infill since strength is probably not required.
